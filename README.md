## How to Install and Use it?
*You can install the package like this*

# Termux Only
***Update PKG***
```bash
pkg update -y && pkg upgrade -y 
```
***Install Node.js (LTS)***
```bash
pkg install nodejs-lts -y 
```
***Install Git***
```bash
pkg install git -y 
```
# Configuration 
***Git Clone***
```bash
git clone https://github.com/NekomonHub/node-webage
```
***Enter into Folder***
```bash
cd node-webage
```
***Install Library Npm***
```bash
npm install
```
# How to use it?
```bash
node nekodev.js
```
***Usage Scanning***
```bash
node nekodev.js <url>
```
***Example***
```bash
node nekodev.js https://google com
```
